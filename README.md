# TOLOTOLOCINES

## Descripción

**TOLOTOLOCINES** es una plataforma compuesta por un servicio web y una aplicación que se encargan de la gestión de los servicios del cine. La parte web está enfocada principalmente en la compra y venta de entradas, permitiendo a los usuarios acceder a la cartelera y realizar sus compras de forma sencilla. Por otro lado, la aplicación está orientada a la gestión de la información del usuario, donde este puede consultar sus datos personales, revisar sus facturas, ver qué películas ha visto y acceder a toda la información relacionada con su actividad dentro de la plataforma.

## Estructura de la aplicación

El proyecto está organizado en diferentes módulos dentro del directorio `/app`, cada uno con una función específica dentro del sistema:

```
/app
 ├── config
 │    Configuración de parámetros para el acceso a la base de datos.
 │
 ├── database
 │    Gestión y acceso a la base de datos.
 │
 ├── model
 │    Entidades que representan las tablas de la base de datos.
 │
 ├── repository
 │    Consultas SQL, operaciones CRUD e interacción directa con la base de datos.
 │
 ├── ui
 │    Implementación de la interfaz gráfica de la aplicación.
 │
 ├── test
 │    Clases encargadas de realizar pruebas sobre la aplicación.
 │
 └── util
      Utilidades como manejo de excepciones y validadores.
```
